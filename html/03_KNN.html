KNN¶

El Nearest Neighbors es un tipo de aprendizaje basado en instancias o aprendizaje no generalizado : no intenta construir un modelo interno general, sino que simplemente almacena instancias de los datos de entrenamiento. La clasificacin se calcula a partir de un voto de mayora simple de los vecinos ms cercanos de cada punto: a un punto de consulta se le asigna la clase de datos que tiene ms representantes dentro de los vecinos ms cercanos del punto
